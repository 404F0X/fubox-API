# New API MVP Priority Plan

同步日期：2026-06-07

## 产品口径

当前产品主线收敛为 New API 风格的多用户 API 分发平台，不是商户支付平台。目标不是长期停留在 operator handoff packet，也不是先做企业级 SSO/支付/订阅，而是先跑通非商户三层：

```text
站长/管理员 -> 配置渠道、模型、价格、兑换码、用户、日志
用户/开发者 -> 注册登录、兑换额度、创建 API Key、调用 OpenAI-compatible API、查看余额和用量
上游供应商 -> OpenAI / Anthropic / Gemini / OpenRouter / 自定义渠道
```

AxonHub 方向保留在体验层：Any SDK -> Any Model、模型别名/Profile、Trace、协议兼容和排障解释。

## P0：New API MVP 当前主线

目标：用户不需要理解内部 handoff packet，也能像 New API 一样登录、拿额度、创建 API Key、调用接口、查看用量。

   - Status：Implemented / verified。

   - DoD：支持本地账号注册/登录/退出；用户 session 与 tenant/project/user 绑定；不依赖外部 OIDC。
   - Scope：`apps/control-plane`、`db/migrations`、`web/admin-ui`。
