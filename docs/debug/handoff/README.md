# Debug Handoff Notes

This directory is for historical operator/debug handoff material that should not sit on the primary Admin UI path.

Use it for:

- proof-shaped notes that explain old evidence artifacts
- command or artifact handoff context
- migration breadcrumbs for legacy UI cleanup

Do not use it to create new production gates for current P0/P1 frontend work. Main UI copy should stay focused on status, reason, and next action.
